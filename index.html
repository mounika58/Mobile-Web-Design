JavaScript

JavaScript is a CS (Client Side) scripting language. It is in no way related to Java as it's name suggests, but it is used for client side scripting. By Client Side scripting, i mean the ability to manipulate data, the DOM and a webpage as a whole, without having a backend service or language.

Bootstrap

Bootstrap is a front-end framework which includes HTML and CSS based design templates for typography, forms, buttons, navigation, and many other, as well as JavaScript plugins. Bootstrap also gives you the ability to easily create Responsive web design, it is about creating web sites which automatically adjust themselves to look good on all devices,from small phones to large desktops.
